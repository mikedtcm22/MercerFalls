# DM-OS — Pre-Campaign Timeline

Up: [[DM-OS & Prep]]

**The T-4 → T-0 onboarding project plan (May 9 → June 9, 2026 Session 0). Each week has a build focus, a decide focus, and a player-facing artifact that builds anticipation and warms up the capture pipeline before play starts.**

> [!note] Project-scoped, retires at Session 0
> Unlike the recurring cadence in [[DM-OS — v0.1 Specification]], this is a finite project. Its lessons get reviewed in the per-Act heavy consolidation (which pre-campaign artifacts paid off most). Status provisional — recruitment realities can compress the schedule.

## Timeline Overview

| Week | Dates | Build focus | Decide | Player artifact |
|---|---|---|---|---|
| **T-4** | May 9–15 | Recruitment + tech infrastructure | Final tech stack; cancellation threshold | Atmospheric teaser paragraph |
| **T-3** | May 16–22 | Onboarding materials + Discord setup | Cast finalized | Char creation guide + town primer |
| **T-2** | May 23–29 | Char creation 1-on-1s; pipeline dry run | Tone-customization first iteration | Per-character "last summer week" vignettes |
| **T-1** | May 30–Jun 8 | Session 1 fully prepped; recap pipeline drilled | Final operational details | *Mercer Falls Tribune* front page |
| **T-0** | June 9 | **Session 0** | — | Session itself + first Dispatch next day |

Recruitment runs in parallel across T-4/T-3; Session 0 readiness is gated on cast lock.

## Week-by-Week

- **T-4 — Recruitment & infrastructure.** Sharpen per-recruit pitch language; open parallel conversations (filter for tone fit, not D&D experience); stand up the Discord server skeleton (`#out-of-character`, `#in-character`, `#bot-commands`, `#handouts`, per-character private channels); create the Archivist campaign and draft the **tone-customization prompt** (third-person, present tense, small-town-horror register); import bible/char-guide/[[The Bizarro Plot]] into Private Journals; wire up the Discord–Archivist `/start`/`/end` integration. *Artifact:* a ~100-word "Mercer Falls feels like…" teaser (also a recruitment tool — see [[The Campaign Pitch]]).
- **T-3 — Cast finalization & onboarding kickoff.** Lightly revise the char creation guide for the actual cast; draft **initial NPC briefs** in the bible (diner owner Marge, principal, librarian, [[Henry Voss]], pharmacist, key teachers, PC parents/siblings); draft the town primer ("What you know about Mercer Falls"); set up Archivist **Find & Replace** rules to protect proper nouns; lock the calendar. *Artifact:* char creation guide + town primer, pinned in `#handouts`.
- **T-2 — Character development (light week / Memorial Day).** ~30-min one-on-one character creation calls; draft the [[Session 0]] outline and a rough [[Session 1 — The Dare|Session 1]] outline; run the **capture pipeline dry run** ("Session 0.5" test recording) and iterate tone customization. *Artifact:* per-character "last summer week" vignettes (the highest-leverage chemistry artifact) + optional AI yearbook portraits and Archivist trading cards.
- **T-1 — Final prep.** Session 1 fully prepped (cold open written, monster statted, locations and NPC voices ready); recap pipeline mentally drilled; **Cold open #1** written; **Avrae** configured with per-PC sheets. Pin all v0.1 conventions in Discord. *Artifact:* the **Mercer Falls Tribune** front page for **Sunday, August 25, 1996** (Session 1's in-fiction date) — local news, weather, police blotter (incl. a stale "missing person"), community calendar, a Letter to the Editor about the avoided Highway 21 stretch, classifieds. **One detail is subtly wrong** — the seed for the Session 1 cold open.
- **T-0 — Session 0 (June 9).** Introductions, in-fiction character intros, a mundane opening scene, a closing beat pointing toward Session 1's hook. The Wednesday after, the first Mercer Falls Dispatch goes out. The system is now live.
