# Source Documents

Up: [[Embers of the Gathering Hollows — Overview]]

This vault is the **integrated, atomized layer** over the campaign's long-form source documents. The originals remain the authoritative record and live one level up, in the parent `D&D - Mercer Falls/` folder (outside this vault). Each vault note's `source` property points to the document and section it derives from. When canon changes, update the source doc *and* the affected notes.

## Other artifacts (not yet vault notes)
The parent folder also holds the campaign trailer, cover/clique art, the website plan, and an `(archive)/` of superseded versions. These are production assets rather than canon; reference them as needed but they are out of scope for the wiki layer.

## Versioning note
Filenames carry version numbers (bible `v0_4`, town `v0_3`, bizarro `v5`). When a new version supersedes one, move the old file to the parent `(archive)/` folder and refresh the affected notes' `source` properties.
