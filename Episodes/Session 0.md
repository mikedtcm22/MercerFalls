# Session 0

Up: [[Episodes]]

**Pre-play onboarding: D&D basics, character creation, the era/setting, and a full teach of the Dungeoneering Stretch system — then an optional soft open into the dare and the awakening.** Target length 2–2.5 hours; hard requirement is only Block A, with Blocks A–C the realistic stopping point and A–H the stretch goal.

## What to cover (bible §19)
- **D&D basics** if anyone's new.
- **Character creation** — junior-high / high-school age, summer break 1996 (see [[Character Creation Guide]]).
- **Era and setting at a high level** — 1990s [[Mercer Falls, Ohio]]; you all live here; *no magic exists in your world as far as anyone knows.*
- **The [[Dungeon Mechanics|Dungeoneering Stretch system]] in full** — table-heavy, taught openly, not surprise-dependent.
- **Marching order and the Crawl Strategy concept.**
- Introduce the **[[Time Points]]** framework as a concept; defer the detail to Session 1.5.

## What NOT to explain
Discovered through play, not taught here:
- Roguelike daily resets
- The full Time Points system (Session 1.5)
- The dissolve / [[Hollowing and Agency-Loss|hollowing]] mechanic
- The [[The Reality Filter|Reality Filter]] as a phenomenon
- Anything about the dungeon's nature, Wesley's role as its user, the [[Order and Chaos|cosmology]], or the embers' real name

## Soft-open block sequence (session-0_outline.md)
| Block | Content |
|---|---|
| A — Setting & Mechanics | Pitch, 1990s vibe, char-creation flavor, the Stretch system, Time Points concept |
| B — Character Creation | Collaborative; class-flavored hooks foreshadow the Awakening; DM notes each PC's evocative detail |
| C — Party Assembly | "Do you know each other? How?"; anchor relationships; *most likely stopping point* |
| D — Cold Open | The drifter cooking beans, the knocked can, the woods going still — foreshadows [[The Hush]]'s sound-MO |
| E — The Dare | Bullies, gate, padlock, sunrise return; [[Ricky Vance]] present; ask each PC why they accept |
| F — The Camping Lull | Spotlight moments; mill grounds; tension; closes on a screech |
| G — The Chase | Funnel to the blast furnace; environmental beckoning; end at the open hatch |
| H — The Awakening | Antechamber, the wall warms, embers + equipment niche; cliffhanger on the chamber closing |

If Session 0 runs long it stops at Block C; the dare/chase/awakening then open [[Session 1 — The Dare]]. The bible (§16) notes the standing tabled idea of splitting Session 1's content across a longer Session 0 plus a shorter Session 1 — decision deferred.
