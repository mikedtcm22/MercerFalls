# Embers of the Gathering Hollows — Overview

> Home note for the Mercer Falls campaign vault: what the campaign is, how this vault is organized, and where to start. Living index — each domain below has its own MOC that links out to the atomic notes. Built to mirror the Arachne-style second-brain layout (see [[Vault Setup & Conventions]]) so an LLM (and you) can reach any piece of canon in one or two hops.

## What the campaign is

A **D&D 5e roguelike-megadungeon campaign with *Persona*-style in-town time mechanics**, set in **Mercer Falls, Ohio, summer 1996** — a dying rust-belt town where magic is supposed to be impossible. Kids stumble into a century-old infernal machine beneath the town that **eats agency** (presenting to the world as time-loss), and are saved by **embers** granted by the thing the machine was built to contain. Tonal touchstones: *Buffy*, *Stranger Things* (deliberately diverged), *Persona*. Act 1 runs Levels 1–5, mid-June through the **autumn equinox (Sept 22, 1996)**.

The deepest argument of the campaign: **agency is the answer** — against both cosmic forces that want choice removed. See [[High Concept]] and [[Order and Chaos]].

## How to read this vault

Start here, then follow links. The navigation backbone is **Maps of Content (MOCs)** — one per domain — not folders. `[[Wikilinks]]` resolve by note name regardless of folder, so move notes freely. Every note carries YAML **properties** (`type`, `domain`, `status`, `visibility`, `tags`, `source`) and an `Up:` link to its MOC. DM-secret material is flagged two ways: a `visibility: dm-only` property and an inline `> [!danger] DM-only` callout, so the future MCP connector can filter what a player-facing LLM is allowed to surface. See [[Vault Setup & Conventions]].

## Conventions in one breath

A note earns its place when it's an **atomic, linkable entity** — an NPC, location, faction, system, plot thread, episode, or piece of lore. Notes summarize and interlink; they point to [[Source Documents]] for exhaustive detail rather than duplicating it. Title Case names; `Up:` to the domain MOC; DM-secrets flagged.
